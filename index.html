Exploratory Data Analysis of COVID 19 Clinical Trial Data

This project explores publicly available COVID-19 clinical trial data obtained from Kaggle to understand global research trends during the pandemic. The analysis focuses on study status, participant demographics, enrollment patterns, and study outcomes. The goal of this project is to demonstrate data cleaning, exploratory analysis, visualization, and interpretation skills using Python.

Exploratory Data Analysis of Colorectal Cancer Data

This project presents an exploratory data analysis (EDA) of a colorectal cancer dataset. The objective is to examine demographic and clinical patterns in age distributions, Dukes cancer stages, and survival after treatment of colorectal cancer.
