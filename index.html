DNA Sequence Analysis and Transformation Tool (Python)

This project implements a Python script for analyzing DNA sequence data stored in FASTA format. It parses raw nucleotide sequences, excluding metadata, and computes key genomic metrics including base counts, GC and AT content percentages, and ambiguous nucleotide (N) frequency. The script also generates the reverse complement of the sequence and performs DNA-to-RNA transcription. The project demonstrates practical bioinformatics skills, efficient algorithm design, and a strong understanding of molecular biology fundamentals applied in computational workflows.

Traffic Source Performance Analysis for Digital Advertisements

This project analyzes user engagement with online advertisements using Python and Pandas. The objective is to identify which traffic sources generate the highest number of clicks and to compare performance differences between experimental ad groups using A/B testing principles. The workflow includes data cleaning and transformation, data aggregation using GroupBy and pivot tables, boolean filtering, and statistical comparison to support data-driven advertising decisions.

E-commerce Conversion Funnel & Time-to-Purchase Analysis

This project analyzes user behavior through an e-commerce conversion funnel for Cool T-Shirts Inc.. Using Python and Pandas, I examine how users move from visiting the website to adding items to their cart, proceeding to checkout, and completing a purchase. The analysis identifies funnel drop-offs and calculates the average time taken from initial visit to purchase.

Jeopardy! Data Analysis with Pandas

The analysis focuses on data cleaning, text processing with regular expressions, and exploratory analysis, including filtering questions by keywords and computing summary statistics.The project demonstrates practical pandas skills such as column standardization, string manipulation, numeric conversion, and basic analytical queries.
